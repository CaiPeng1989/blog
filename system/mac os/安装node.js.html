# mac os 下安装node.js

前提

干净的Mac OS系统

  1. 使用 n-install 安装 n 以及 node.js。

执行以下命令：

sudo curl -L https://git.io/n-install | bash -s -- -y latest
1

TIP

latest： node.js最新版本，你也可以替换为lts（长期支持版本）或者具体的版本号（例如：12.7.0）
