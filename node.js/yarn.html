# 修改 yarn 镜像源

TIP

镜像源列表

TIP

以下方法任选其一

  • 利用 yarn config set 命令修改。
yarn config set registry 镜像源
1
  • 利用 yrm 修改。查看 nrm 和 yrm 的区别
npm i yrm -g
yrm ues 镜像源别名
1
2
  • 直接修改 .yarnrc 文件（仅适用于windows）。打开 系统盘符:\Users\当前登录用户名 目录，找到/新建 .yarnrc 文件并修改内容。本人.yarnrc 文件内容。
registry "镜像源"
1

# 使用 yarn 全局安装 vue 脚手架

  1. 全局安装vue脚手架
yarn global add @vue/cli @vue/cli-init
1
  1. 尝试执行vue命令，如果报错的话需要 将 yarn的安装位置添加到环境变量
vue -V
