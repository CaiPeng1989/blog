# 修改npm镜像源

TIP

镜像源列表

TIP

以下方法任选其一

  • 利用 npm config set 命令修改。
npm config set registry 镜像源
1
  • 利用 nrm 修改。查看 nrm 和 yrm 的区别
npm i nrm -g
nrm ues 镜像源别名
1
2
  • 直接修改 .npmrc 文件（仅适用于windows）。打开 系统盘符:\Users\当前登录用户名 目录，找到/新建 .npmrc 文件并修改内容。本人.npmrc 文件内容。
registry=镜像源
1

# 使用npm全局安装yarn并添加环境变量

  1. 首先当然是全局安装 yarn。修改npm镜像源
npm i yarn -g
1
  1. 找到/获取 yarn 全局接安装到已安装的可执行文件的位置，默认情况下将是：系统盘符:\Users\当前登录用户名\AppData\Local\Yarn\bin。
yarn global bin
1

  1. 将安装位置添加到环境变量 PATH 并重新打开 shell 即可食用。
